Crimild
=======

A scene graph based framework for interactive 3D applications

Requirements
============

While Crimild is written is pure C++, it depends 

* CMake 
* GTest

Building for MacOS
==================

Requirements:
* CMake 2.8.10.2
* GTest 1.6 (optional)
